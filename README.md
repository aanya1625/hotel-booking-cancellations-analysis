# 🏨 Hotel Booking Cancellations Analysis

## Overview
An Excel-based analysis of 119,390 hotel booking records to understand 
cancellation patterns across City and Resort Hotels (2015–2017).

## 📊 Dataset
- **Records:** 119,390 bookings
- **Hotels:** City Hotel & Resort Hotel
- **Period:** 2015 – 2017
- **Source:** Hotel booking demand dataset

## 🔍 Key Findings
- Overall cancellation rate: **37%**
- **Couples** make up the largest guest segment (68%)
- City Hotels have higher cancellation rates than Resort Hotels
- Room type mismatch is a significant cancellation indicator
- Top booking countries: Portugal, UK, France, Spain, Germany

## 🛠️ Tools Used
- Microsoft Excel (Pivot Tables, Slicers, Dashboard)

## 📁 File Structure
hotel_booking_cancellations.xlsx
├── hotel booking cancellations  ← raw data
├── pivot                        ← pivot table analysis
└── dashboard                    ← interactive dashboard
